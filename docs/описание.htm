Дизайн-документ: общее описание

Жанр:

Жанр: игра включает элементы экономического симулятора, боевые действия главного героя и развитие персонажа.

Геймплей:

Весь геймплей состоит из след.частей:
  • Управление отдельным персонажем (аватаром).
      Игра аватором включает следующие виды:
    • Полет в космическом пространстве, управляя кораблем.
    • Похождение по планете персонажем-аватаром.
  • Стратегическое управление имеющимися ресурсами.
  • Экономика: ограниченные ресурсы, производимые товары, торговля.

Профайл игрока (Account):

    Свойства:

  • Password - пароль.

  • Money - количество денег.

Аватар (Avatar):

Определяет персонажа игрока.
    Свойства:
  • Nick - игровое имя аватара.
  • HP - кол-во жизней.
  • AccountID - id аккаунта, которому принадлежит аватар
  • ZoneID - id зоны, в которой находится игрок
  • Level - Уровень.
  • SkillHolderMap - массив навыков, опыт по каждому из них.
  • ItemHolderMap - массив предметов, принадлежащих только аватару.

Вещь (Thing):

Описывает базовую абстрактную сущность, у которой определены след. свойства:
    Свойства:
  • Name - имя.
  • Description - подробная информация, описание.
  • BasePrice - базовая цена.
  • BodyStrength - кол-во жизней корпуса

Корабль:

    Свойства:
  • HyperSpaceJumpRange - дальность межзвездного гиперпрыжка.
  • HyperGalacticJump - возможность межгалактического гиперпрыжка.
  • FuelHolder - количество и тип топлива для гиперпрыжков.
  • FuelAmountMaximum - максимальное количество топлива.
  • EquipmentHolderMap - Установленное оборудование
На каждое место установки можно установить несколько единиц оборудования.
    Типы устаналиваемого оборудования:
  • Вооружение.
  • Защитное.
  • Двигатель.
  • Навигационное.
  • Дополнительное.
    Типы кораблей (различаются максимальным количеством устанавливаемого оборудования).
  • Легкий боевой истребитель.
  • Средний боевой истребитель.
  • Тяжелый боевой истребитель.
  • Легкий транспортный корабль.
  • Средний транспортный корабль.
  • Тяжелый транспортный корабль.
  • Корабельные платформы.

Топливо:

    Виды топлива:
  • смесь водорода и кислорода.
  • топливо для ядерных реакторов.

Оборудование:

    Свойства:
  • TargetSetInType - тип категории вещей, в которых может устанавливаться оборудование.
  • EnergyConsume - количество потребляемой энергии при работе.
  • Producer - производитель. Производителем может быть планета, звезда, или созвездие.
  • Deterioration - износ 0% работает во всех случаях, 100% - нифига не работает.

Общее оборудование:

  • Вооружение.
      Свойства:
    • Radius - радиус действия
    • MinimumDamage, MaximumDamage - повреждения
    • RechargeTime - время перезарядки
    • HeatTime - время нагрева
      Существуют следующие типы вооружения:
    • лазерное
    • ракетное
    • бластерное
    • бомбы (мины)
  • Защитное.
    • энергетический щит: с регенерацией, без регенерации.
    • противоракетная система.
  • Двигатели.
      Типы:
    • двигатель, работающий на водородно-кислородном топливе.
    • двигатель, работающий на радиоактивном топливе.
    • двигатель, работающий на энергии.
  • Бортовой компьютер.
    • автопилот.
    • система автоматической стыковки со станцией.
    • система автоматической посадки на планету.
    • Навигационная система.
    • Радар.
    • Дополнительное:
    • установка Hyper-drive. Система ускорения полета.
    • система Hyper-Space. Система для межзвездного прыжка.
    • система Hyper-Galactic. Система для межгалактического прыжка.
    • грузовой отсек.
    • спасательная капсула.
    • система регенерации энергии щитов.
    • общий энергогенератор. Виды энергогенераторов: на солнечных батареях - (в системах с активными звездами) ядерные, атомные, нейтронные - (тратятся различные ресурсы)
    • ускоритель регенерации энергии энергогенератора.
    • система сбора газообразных ресурсов из атмосферы планеты или звезды и преобразования газообразных ресурсов в топливо.

Оборудование орбитальных станций.

К данному типу относится оборудование, устанавливающиеся в орбитальных станциях.
    Виды:
  • Генератор кислорода. Производит кислородный воздух, т.е. есть возможность высадки на станцию без скафандра.
  • Топливный генератор. Из количества ресурсов производит 1 единицу топлива. Необходимо для заправки кораблей.
  • Различное исследовательское оборудование.

Планетное оборудование.

Оборудование данного типа может быть построено на стационарных объектах: планеты, астероиды.
    Виды:
  • Производственное оборудование. Производит из набора ресурсов один товар.
  • Для добычи ресурсов. Добывает кол-во ресурсов в ед.времени.

Космическое оборудование.

В эту категорию входят различные системы и устройства, которые работают без участия человека в открытом космосе.
    Виды:
  • Электронные (шпионские, навигационные) спутники.
  • Исследовательские зонды.

Виды действий, возможных для совершения персонажем/игровым объектом:

    Виды:
  • 1 перейти
  • 2 разрушить
  • 3 купить
  • 4 подобрать
  • 5 построить
  • 6 срубить
  • 7 посадить
  • 8 ландшафт

Производство:

    Базовые принципы:
  • Товары проиводятся из определенного количества других товаров при помощи специального оборудования с затратом определенного количества энергии.
  • Для производства различных объектов необходимы специальные лицензии: например, для покупки/продажи станций, астероидов и т.д.
  • Производство может быть создано в любом стационарном месте (планета, спутники, орбитальные станции, астероиды).
  • Существуют различные ненужные товары, оборудование можно разобрать на ресурсы. Для производства единицы товара необходимо затратить какое-нить количество энергии и набора ресурсов, с выделением кол-ва "Промышленных отходов"
    Сущности:
  • Игровой объект в разных технологических процессах может выступать в роли ресурса или товара, т.е. либо быть затраченным при производстве, либо произведенным, в ходе процесса.
  • Игровой персонаж/объект, кто стартует процесс
  • Технологический процесс (technology_process):
    • Действие, необходимое для начала процесса, данным действием должен обладать персонаж
    • Время, требуемое на запуск процесса, запуск процесса осуществляется персонажем
    • Время, требуемое на процесс
    • Деньги, требуемые на процесс (должны иметься перед запуском процесса)
    • Объекты, произведенные (produced) в ходе процесса
    • Объекты, затраченные (spended) в ходе процесса (затраченные ресурсы, должны иметься в наличии у персонажа)
    • Параметры (options):
      • 0 - автостартующий процесс, при появлении требуемых gok
      • 1 - процесс, выполняющийся один цикл, удаляющий объекта, выбрасывающий созданные gok наружу
      • 2 - процесс, выполняет циклы, до тех пор пока требуемые gok есть в наличии, как только они заканчиваются, объект удаляется
    Технологические процессы
  • Срубить дерево
  • Построить объект
  • Посадить растение

Торговля:

Цены устанавливаются на различные объеты по разному: из единого регистрирующего центра, от спроса/предложения, от монополии различных государств на товары, от редкости объекта.

Виды торговых товаров:

  • Напитки: газированные напитки, легкие алкогольные напитки, крепкие алкогольные напитки, фруктовый сок
  • Одежда: легкая одежда, теплая одежда
  • Ручное оружие: огнестрельное, лазерное, бластерное
  • Высокотехнологичные товары: компьютеры,
  • С/х машины: тракторы,
  • Предметы роскоши: меха, драгоценные камни, украшения
  • Отходы: радиоактивные отходы, промышленные отходы
  • Запрещенные товары: наркотики, рабы
  • Предметы внеземных цивилизаций

Виды ресурсов:

  • Зерно, Молоко, Мясо, Рыба, Фрукты, Овощи
  • Растительный белок, Животный белок
  • Железо, Уголь, Камни, Дерево, Нефть
  • Пластик, Титан, Алюминий
  • Золото, Серебро, Палладий, Платина, Медь
  • Плутоний, Уран

Обитатели

  • едином космическом пространстве существуют различные рассы.
  • каждой рассы есть отличительные особенности, характерные черты, среда обитания, умения (даже оборудование и корабли) и т.д.
  • Некоторые рассы ведут меж собой войну, а некоторые в союзе.

Правление.

  • Существуют несколько "межгалактических союзов", корпораций, альянсов.
  • На каждой населенной планете установленно правление какого-либо союза, либо если власть никому не принадлежит, то анархия.
  • Для установки власти необходимы специальные лицензии, разрешения. Установку проводят по специальным соглашениям.

Квесты.

  • Четкой сюжетной линии нет. Есть различные задания, которые повышают общий опыт, статус, добавляют денег и счастья :)). Количество положительного эффекта будет зависеть от времени, которое дается для выполнения задания, и за которое выполнилось задание.
  • Задания можно получить как от правительств и организаций, так и отдельных каких-либо граждан.
  • Все задания можно собирать и выполнять не пооочереди, а в любом порядке.
